# AulaEstudiantilApp

Aplicación móvil desarrollada en Android que permite gestionar información académica de estudiantes. 
La aplicación integra diferentes módulos para facilitar la consulta de tareas, calificaciones, pagos y comunicación dentro del sistema.

## Descripción del Proyecto
AulaEstudiantilApp es una aplicación diseñada para centralizar la información académica en un solo lugar. 
Los usuarios pueden iniciar sesión y acceder a diferentes funciones del sistema como consulta de tareas, calificaciones y pagos escolares.

## Tecnologías Utilizadas
- Android Studio
- Kotlin
- XML para diseño de interfaces
- Material Design
- Gradle

## Funcionalidades
- Inicio de sesión de usuario
- Consulta de tareas
- Visualización de calificaciones
- Gestión de pagos
- Sistema de chat
- Información institucional

## Estructura del Proyecto
El proyecto se encuentra organizado en diferentes carpetas:

- **app:** contiene el código principal de la aplicación.
- **res:** contiene recursos visuales como layouts, colores e imágenes.
- **activities:** contiene las diferentes pantallas del sistema.
- **gradle:** configuración del sistema de compilación.
